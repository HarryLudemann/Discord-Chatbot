# Hazzahs Discord-Bot
## Install Dependencies:
###### **Conda**:
```
conda create --name <env> --file condarequirements.txt
```
###### **Pip**:
```
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
```
## Create .env:
Create a .env file in the main directory, add your discord bots token. eg.
